Argo CD 3.0 — Small but mighty — Photo Credit Regina Voloshin We are very excited to announce an important milestone for Argo with the Argo CD 3.0 release candidate ! This is the first major version change to Argo CD since 2021. Argo CD 3.0 brings improvements to security, performance and distills dozens of best practices to provide better defaults while still allowing for flexible configuration to match any team.
